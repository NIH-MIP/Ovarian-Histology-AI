These are Jupyter Notebooks to inference whole slide images (WSI)'s in digital pathology to detect and quantitate follicles in human ovarian tissue. Simply install python and Jupyter Lab, download the notebooks and run them. 

1. The scripts expects the model weights to be in a subdirectory called /models within the same directory as the notebooks
2. You will need to install the necessary python packages and libraries to run the notebooks
3. Model weights are available at https://zenodo.org/records/17187832
